# Тест-кейсы: sberbusiness.live

**Объект тестирования:** поиск по сайту, форма подписки на рассылку  
**Окружение:** Chrome последней версии, macOS, разрешение 1440×900

---

## TC-01: Поиск по валидному запросу

| Поле | Значение |
|---|---|
| **ID** | TC-01 |
| **Название** | Поиск возвращает результаты при вводе валидного запроса |
| **Предусловия** | Открыт сайт sberbusiness.live, пользователь не авторизован |
| **Шаги** | 1. Нажать на иконку поиска (лупа) в шапке сайта <br> 2. Ввести в поле запрос «Финансы» <br> 3. Дождаться появления результатов |
| **Ожидаемый результат** | Появляется список релевантных статей. Отображается ссылка «Показать все результаты» |
| **Фактический результат** | Список статей по запросу «Финансы» отображается корректно |
| **Статус** | ✅ Passed |

## TC-02: Поиск с вводом спецсимвола

| Поле | Значение |
|---|---|
| **ID** | TC-02 |
| **Название** | Поиск корректно обрабатывает ввод спецсимвола `$` |
| **Предусловия** | Открыт сайт sberbusiness.live, модальное окно поиска открыто |
| **Шаги** | 1. Нажать на иконку поиска в шапке <br> 2. Ввести символ `$` в поле поиска <br> 3. Подождать результат |
| **Ожидаемый результат** | Поиск завершается в течение 3–5 секунд. Отображается сообщение «Ничего не найдено» или пустой результат |
| **Фактический результат** | Отображается индикатор загрузки «Ищем подходящие публикации». Поиск не завершается более 1 минуты |
| **Статус** | ❌ Failed |
| **Баг-репорт** | [BUG-01](bug-reports.md#bug-01) |
