## Обо мне

Привет! Меня зовут Инесса, я начинающий QA-специалист.

Прохожу обучение по направлению QA в Школе 21 (Сбер) — проектная методология,

тест-дизайн и инструменты.

В этом репозитории — моя тестовая документация: чек-листы, тест-кейсы и
баг-репорты, выполненные в рамках обучения и pet-проектов.

## Навыки и инструменты

- **Тестирование:** ручное тестирование, функциональное тестирование,
  тест-дизайн (классы эквивалентности, граничные значения), составление
  тест-кейсов, чек-листов и баг-репортов
- **Инструменты:** Postman (базовый уровень), Charles (базовый уровень),
  Git / GitHub, терминал (bash)
- **Трекеры:** Jira / YouTrack
- **Базы данных:** SQL — начальный уровень (SELECT, WHERE, JOIN)
- **Браузер:** Chrome DevTools (вкладки Network, Console)

### Проект 1: Тестирование сайта sberbusiness.live

Функциональное тестирование публичного медиа-ресурса для предпринимателей.

**Что сделано:**
- Чек-лист проверок поиска и формы подписки
- Тест-кейсы по двум функциональным блокам

### Проект 2: Тестирование платформы аналитики медиаконтента

**Что сделано:**
- Проверка ключевых сценариев: добавление аккаунта, обработка видеоконтента
- Выявлена критичная проблема с транскрибацией длинных роликов — функциональность доработана до запуска
- Оформлен баг-репорт по визуальному дефекту

**Инструменты:** Chrome DevTools, ручное тестирование
